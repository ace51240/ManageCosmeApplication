# コスメ管理アプリ

このアプリは手元にあるコスメを登録し、楽天やamazonなどの購入ページのリンクを記載したり、複数のコスメをメイクレシピとして登録するアプリです。

## 概要
このプロジェクトはAndroidとiOSのアプリをKMPで実装するプロジェクトです。

## 機能要件

### Must
- 手元にあるコスメをアプリに登録できます。
   以下の方法で登録できます。 
    - 手動登録
       - 手入力でコスメ情報を入力して登録できます。
    - 写真登録
       - スマホで撮影した写真やギャラリーから選択された写真のみを使ってコスメを登録できます。
    - AI登録
       - スマホで撮影した写真やギャラリーから選択された写真から、手入力相当の情報を解析してコスメを登録できます。
- 登録したコスメを編集・削除できます。
- 登録したコスメを各ジャンルごとに分類できます。 
- 登録したコスメをお気に入り登録できます。
- 登録したコスメから楽天やAmazonのショッピングサイトをWebページで開くリンクボタンが表示されます。 
- 登録したコスメを複数集めてメイクレシピを登録できます。
- マネタイズ用に広告収益をいれます。

### Want
- GoogleアカウントやLineアカウントなどを使ってログインできます。
- AI登録で取得できた情報が本当にコスメ情報として正しいのか、サーバー側でバリデーションを行います。
- 複数の端末でコスメ情報を共有できるよう、引継ぎの設定を追加します。
- 広告収益以外の有料オプションを利用できます。

## 注意事項
- 写真はセンシティブな情報を含んでいるため、サーバーにアップロードせず、アプリ内で消化できるようにします。
- AI解析はスマホのみで行います。

## UI/UX

### 🔹 UX全体構造（ユーザーフロー）

```
[ホーム]
   ├ 登録ボタン（＋）
   │   ├ 手動登録
   │   ├ 写真登録
   │   └ AI登録（将来拡張）
   ├ コスメ一覧（タブ：全て／ジャンル／お気に入り）
   │   └ コスメ詳細（編集／削除／リンクボタン）
   └ メイクレシピ一覧
       └ レシピ詳細（構成コスメ／メモ）
```

### 💡 UXデザイン方針

| ポイント              | 意図                     |
| ----------------- | ---------------------- |
| 登録導線を最短に          | 「＋ボタン」から2タップ以内で登録開始できる |
| 分類・検索を直感的に        | ジャンルタブ＋お気に入りスターで切替     |
| 編集・削除を軽く          | スワイプ or 長押し操作で可能       |
| メイクレシピを“作品”として見せる | カード型UI＋サムネイルを強調        |
| AI登録と手動登録の見た目を統一  | “入力補助”として自然に統合         |

## 🔹 ユーザー体験シナリオ（例）

> Aさんがリップを買ったとき：

1. アプリを開く → ＋ボタン → 「写真登録」
2. パッケージを撮る → OCRで「資生堂 ルージュ」検出
3. 補完候補から選んで登録完了（10秒）
4. ホームに反映 → お気に入り登録
5. 次回のメイクレシピ作成時にそのリップを選択

→ “自分の持ち物が自動で整理されていく”快感を狙うUXです。

## 🏠 ホーム画面

* **主な遷移先：**

    * ➕ボタン → 「登録モーダル」
    * カードタップ → 「コスメ詳細」
    * 下部ナビ「レシピ」 → 「メイクレシピ一覧」
    * 下部ナビ「設定」 → 「設定画面」

---

## ➕ 登録モーダル

* **選択肢：**

    * 手動登録画面へ
    * 写真登録画面へ
    * AI登録画面へ（将来的に）
* **完了後：** 登録完了ダイアログ → ホームへ戻る

## 📝 手動登録 / 🖼 写真登録

* **遷移元：** 登録モーダル
* **遷移先：** 登録完了 → ホーム画面
* **補足：**

    * 写真登録ではOCR解析完了後、自動入力→編集→保存フロー

---

* **遷移元：** ホーム画面
* **遷移先：**

    * 編集モード（詳細内）
    * 削除（確認ダイアログ）
    * 「楽天で見る」／「Amazonで見る」ボタン → 外部ブラウザ
* **戻り先：** ホーム画面

---

## 🎨 メイクレシピ一覧

* **遷移先：**

* **戻り先：** ホーム画面

## 🎨 レシピ詳細
